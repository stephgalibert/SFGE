# ECGE
Entity Component Game Engine

## Dependencies

* https://github.com/erincatto/box2d/tree/v2.4.1   
* https://github.com/skypjack/entt/tree/v3.6.0
    * copy the src folder into `/usr/include`
  
## Build

* mkdir build && cd build
* cmake ..

## Run

* ./sample
